# Project-Algorithm
## Shortest Path Navigation System

This project implements and compares two shortest path algorithms:
- Dijkstra’s Algorithm
- A* Algorithm

## Features
- Shortest path calculation
- Runtime comparison
- Expanded nodes analysis
- Best / Average / Worst case testing
- Graph visualization

## Technologies Used
- Python
- Google Colab
- NetworkX
- Matplotlib

## Algorithms
### Dijkstra’s Algorithm
Finds the shortest path by exploring nodes with the minimum distance.

### A* Algorithm
Uses heuristic guidance to improve search efficiency.

## Project Goal
Compare the performance of Dijkstra and A* algorithms in navigation systems.
